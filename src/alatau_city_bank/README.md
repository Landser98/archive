# Alatau City Bank - Batch Parser

## Описание

Batch-парсер для обработки выписок Alatau City Bank. Парсит PDF файлы, валидирует данные и вычисляет доходы ИП.

## Использование

```bash
python src/alatau_city_bank/batch_parse.py <path> [опции]
```

### Параметры

- `path` - Путь к PDF файлу или директории с PDF файлами (обязательный)
- `--out-dir` - Директория для выходных CSV файлов (по умолчанию: `<base_dir>_out`)
- `--json-dir` - Директория для JSON метаданных (по умолчанию: `<base_dir>/converted_jsons`)
- `--months-back` - Количество месяцев для расчета доходов ИП (по умолчанию: 12)
- `--no-verbose` - Отключить подробный вывод

### Примеры

```bash
# Обработать один файл
python src/alatau_city_bank/batch_parse.py data/alatau_city_bank/statement.pdf

# Обработать все PDF в директории
python src/alatau_city_bank/batch_parse.py data/alatau_city_bank/

# Указать выходную директорию
python src/alatau_city_bank/batch_parse.py data/alatau_city_bank/ --out-dir results/
```

## Input (Входные данные)

### PDF файлы
- **Формат**: PDF выписки Alatau City Bank
- **Расположение**: `data/alatau_city_bank/*.pdf`
- **Можно указать**: один файл или директорию с файлами

## Промежуточные файлы

### JSON метаданные
- **Путь**: `data/alatau_city_bank/converted_jsons/<stem>.json`
- **Содержит**: метаданные PDF (версия, создатель, даты создания/модификации, структура страниц)
- **Создается автоматически** при первом запуске

## Output (Выходные данные)

Все файлы сохраняются в директории `data/alatau_city_bank_out/` (или указанной через `--out-dir`):

### 1. `<stem>_header.csv`
Заголовок выписки
- Период выписки
- Информация о счете
- Начальный и конечный остаток
- Дата закрытия баланса

### 2. `<stem>_tx.csv`
Транзакции
- Дата операции
- Дебет
- Кредит
- КНП (код назначения платежа)
- Назначение платежа
- Корреспондент

### 3. `<stem>_footer.csv`
Итоговая информация
- Итоговые обороты (дебет/кредит)
- Конечный остаток

### 4. `<stem>_meta.csv`
Метаданные обработки
- `bank` - Название банка (ALATAU_CITY)
- `pdf_file` - Имя исходного PDF файла
- `json_file` - Имя JSON файла с метаданными
- `flags` - Флаги валидации (разделенные точкой с запятой)
- `debug_info` - Отладочная информация в формате JSON

### 5. `<stem>_tx_ip.csv`
Транзакции с флагами ИП
- Все колонки из `<stem>_tx.csv`
- Дополнительные флаги для идентификации доходов ИП
- Пометки об исключениях из расчета дохода

### 6. `<stem>_ip_income_monthly.csv`
Доходы ИП по месяцам
- Месяц
- Сумма дохода за месяц
- Количество транзакций

### 7. `<stem>_income_summary.csv`
Сводка по доходам ИП
- `total_income` - Общая сумма доходов
- `total_income_adjusted` - Скорректированная сумма (с учетом исключений)
- Дополнительная статистика

## Особенности

- Использует Camelot для извлечения таблиц из PDF
- Выполняет числовую валидацию (проверка соответствия сумм)
- Валидирует метаданные PDF (даты создания, создатель)
- Автоматически извлекает КНП из транзакций
- Вычисляет доходы ИП с учетом исключений

## Формат дат

- Формат дат в выписках: `%d.%m.%Y` (дд.мм.гггг)

## Кодировка

Все CSV файлы сохраняются с кодировкой `utf-8-sig` (UTF-8 с BOM) для корректного отображения в Excel.
